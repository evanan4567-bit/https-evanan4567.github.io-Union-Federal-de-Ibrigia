# Unión Federal de Ibrigia (sitio)

## Cómo usar
1) Descomprime el .zip.
2) Sube TODO al repositorio (mismos nombres y carpetas).
3) En Settings → Pages: main + /(root).

## Personalizar partidos
Edita: assets/js/app-config.js
- Cambia name/meta/color.
- Sube tus fotos a assets/img/partidos/
  - partido1.jpg
  - partido2.jpg

## Imagen del congreso
Ya está en: assets/img/congreso.jpg
Puedes reemplazarla por otra con el mismo nombre.
